# BSxFLAT Beta 1.0

based on MODxFlat Manager Theme

https://www.youtube.com/watch?v=Ss0og90n3XM


## Beta 1.0 Features

* Responsive Bootstrap (i)frameset
* Responsive Bootstrap Login
* Responsive Bootstrap Dashboard
* Responsive Bootstrap Menu
